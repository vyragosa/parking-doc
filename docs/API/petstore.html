API для системы бронирования парковки

Получение списка доступных парковочных мест

Возвращает список доступных парковочных мест с поддержкой пагинации, сортировки и фильтрации.

Authorizations:
None
query Parameters
date
required
string <date>

Дата для поиска доступных мест.

start_time
required
string <date-time>

Время начала поиска1.

end_time
required
string <date-time>

Время окончания поиска.

location
string

Локация парковки (необязательный фильтр, может быть на уровне здания, зоны или уровня).

page
integer >= 1
Default: 1

Номер страницы для пагинации.

page_size
integer [ 1 .. 50 ]
Default: 10

Количество элементов на странице.

sort
string
Enum: "available_from" "location"

Параметры сортировки (например, "available_from", "location").

Responses

https://hihihi.com/api/v1/parking-spots
